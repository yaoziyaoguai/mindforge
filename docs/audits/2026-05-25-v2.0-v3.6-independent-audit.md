### P1 (3 items)

#### P1-01: Gate Evidence 不可重现 — pytest exit code=1

- **Evidence**: `pytest tests/ -q` 返回 exit code=1, 2 FAILED。`docs/dev/quality-debt-ledger.md` 声称 "0 (2890 passed, 1 skip)" 无法在当前环境重现。
- **Root cause**:
  - `tests/test_web_api.py:131` — `from docx import Document` 缺少 `pytest.importorskip("docx")` guard（其他测试文件如 `tests/test_v0_2_5.py:338` 和 `tests/adapters/test_optional_dependency_smoke.py:158` 正确使用了 `importorskip`）
  - `tests/relations/test_graph_perf.py::TestPerfDiscoveryContext::test_perf_deterministic_repeat` — flaky test，独立运行 pass，全量 suite 中非确定性失败
- **Why it matters**: 所有 v3.x implementation notes 声称 "pytest exit 0" — gate 不能在标准环境重现，交付真实性存疑。
- **Recommended remediation**: ① `tests/test_web_api.py:129` 添加 `pytest.importorskip("docx", reason="python-docx optional dependency")`；② 调查 flaky test 根因（likely test ordering dependency / shared state）；③ 更新 quality-debt-ledger.md 真实 gate 状态。
- **Suggested owner area**: backend/tests
- **Blocks v3.7?** YES — gate 可信度是继续的前提

#### P1-02: zh-CN user-guide.md 未随 v2.5 更新

- **Evidence**: `docs/en/user-guide.md` (14,351 bytes) 包含 Import/Export/Dogfood/Provider Readiness/Lifecycle/Community 全部章节, 24 处匹配; `docs/zh-CN/user-guide.md` (10,752 bytes) 仅 1 处匹配这些关键词。v2.5 impl note 声称 "更新 user-guide.md" 未区分 locale。
- **Why it matters**: 中文用户看到的文档缺失 v2.5 全部新功能。用户引导与实际能力不一致。
- **Recommended remediation**: 同步更新 `docs/zh-CN/user-guide.md` 或明确在 implementation notes 中声明仅英文版已更新。
- **Suggested owner area**: docs
- **Blocks v3.7?** NO — 不阻塞开发但需在产品发布前修复

#### P1-03: architecture.md 引用不存在的服务文件
