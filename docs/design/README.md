## 文档角色

- **RFC**：方案讨论和决策记录。回答"为什么这样设计"。
- **SDD**：详细设计，面向实现。回答"具体怎么做"。
- **ADR**：架构决策记录。回答"为什么选 A 不选 B"。ADR 位于 `docs/adr/`。
