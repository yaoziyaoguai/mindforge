## 设计红线（全程遵守）

- 不调用真实 LLM / Cubox / Upstage
- 不做 RAG answering / embedding / vector DB
- 不引入大型生产依赖
- 不破坏 ai_draft → human_approved 显式审批语义
- 不处理真实私人资料 / 不写真实 Obsidian vault
- 所有 graph/community 计算均为确定性（无 LLM/embedding）
- API key 不进 Git / YAML / 前端完整展示
