## Historical / Superseded Docs

The early documents in the following directories are historical plans or implementation records and do not represent current product commitments:

- **`docs/plans/`** — Older plans (not the latest active roadmap) are historical.
- **`docs/specs/`** — Historical specification files; implementation may have diverged significantly.
- **`docs/implementation-notes/`** — Notes from 2026-05-24 and earlier are execution records, not current canonical docs.
- **`docs/adr/`** — Some ADRs remain valid, but Graph/Sensemaking-related ADRs have been corrected by the v4.2 truth reset.
- **`docs/design/`** — Design reference documents; do not represent current implementation state.

Documents explicitly marked as superseded can be found in [`docs/dev/docs-reset-index.md`](dev/docs-reset-index.md#superseded-graphsensemaking-docs).

## Lab / Internal Feature Notice

The following features are lab/internal/experimental. They are **not** part of MindForge's main product path, are not exposed in the main navigation, and carry no API stability guarantees:

| Feature | Status | Entry | Notes |
|---------|--------|-------|-------|
| **Graph View** | internal | Library → Graph Explorer (4 NodeType) | Standalone `/graph` route preserved but not in main nav |
| **Sensemaking** | lab | No main nav entry | Experimental analysis (bridge detection, etc. via simple heuristics) |
| **Entity Resolution** | lab | No main nav entry | ConceptCandidate deterministic detection; no auto-upgrade |
| **GraphRepository** | internal | Test-only | Repository Pattern wrapper over GraphPort |
| **Extension Plugin** | lab | No main nav entry | ExtensionManifest/ExportAdapter architecture placeholder |
| **Dogfood Scenarios** | internal | `/dogfood` | Developer/maintainer tools |

The current main product path is:
```
Source / Import → ai_draft → Review → explicit approval → human_approved → Library → Recall (BM25) / Wiki (LLM synthesis) → Export
```

## Documentation Strategy

- **No file moves**: Avoid widespread link breakage.
- **No deletion of historical evidence**: Audits, implementation notes, and ADRs are preserved as project evolution evidence.
- **No rewriting of history**: Old documents may be marked superseded/lab/internal but original content is not modified to the point of distortion.
- **Index-first**: Navigation is provided through this document and [`docs/dev/docs-reset-index.md`](dev/docs-reset-index.md).
